# Goodwin Bricklaying — Website Build

This is the Cursor project for the Goodwin Bricklaying website: a premium, image-led website for David Goodwin, a Melbourne-based masonry specialist with 25 years of experience in heritage restoration, architectural brickwork, and luxury residential masonry. The site is designed to attract high-end homeowners, architects, and developers across inner Melbourne suburbs — and to pre-qualify leads by presenting as a luxury architecture or heritage restoration studio rather than a standard trades website.

**Primary domain:** goodwinbricklaying.com.au
**Redirect:** goodwinbrickblockandheritage.com.au → 301 to primary
**Business email:** info@goodwinbricklaying.com.au
